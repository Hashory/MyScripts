# hashory-scripts

Personal automation scripts.

## Scripts

This package provides the following CLI commands:

- **`install-git-before-delete`**: Runs `install-git-before-delete.ts`
- **`jj-newai`**: Runs `jj_newai.ts`
- **`jj-ai-clean`**: Runs `jj_ai_cleanup.ts`
- **`update-pnpm-global`**: Runs `update-pnpm-global.js`
- **`update-winget`**: Runs `update-winget.ts`

## Setup

```bash
pnpm install
pnpm run setup
```
